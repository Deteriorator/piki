首先什么是神经网络？人脑有上千亿被称做神经元的细胞组成，它们通过突触联结在一起。我们把突触接收刺激并被刺激的过程称作“思考”。

我们可以在计算机上创建神经网络来模拟这一过程。我们不需要在分子级别模拟人脑复杂的生物特征，只需模较高层次的规律。我们使用的数学工具是矩阵，它是由数字组成的网格。为了保持简洁，我们构建的模型只包含一个神经元，三个输入和一个输出。

我们将训练神经元解决下面的问题。前面四行记录被称为训练集。你能发现规律，计算出‘？’是为还是1吗？

你可能已经发现，输出总是和输入的第一列相等。所以‘？’应该是1.

训练过程

但是如何让我们的神经元正确回答问题？我们为每一个输入设置权值，可以为正数也可以为负数。一个输入的权重是一个大的正数，还是一个大的负数，对输出值有很大的影响。在开始前，我们随机设置权值。然后开始训练过程：

  1. 取训练集中取输入数据，修改他们的权重，然后做为参数传输给一个特殊的公式，计算出神经元的输出值。
  2. 计算神经元输出和训练集期望输出值之间的误差。
  3. 根据误差方向，微调权重值。
  4. 重复上述过程10，000次。

最终，调整后的神经元权重将最佳适配训练集。如里输入具有同样特征的新的数据，它会做出很好的预测。

这个过程被称为反向传播。

  2. 如果输出值是一个较大的正数或负数，这意味着神经元对权值很有自信。
  3. 从图四我们看到，较大数值的斜率较小。
  4. 如果神经元非常确认权值的正确性，我们不需要对它做大的调整。乘以Sigmoid的斜率正好可以达到这一效果。
